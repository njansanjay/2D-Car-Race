#  🚗 2D Car Racing Game
![Status](https://img.shields.io/badge/status-complete-brightgreen?style=for-the-badge)

A fast-paced 2D endless car racing game built using HTML, CSS, and JavaScript.
Dodge incoming traffic, switch lanes, and survive as long as possible while your speed continuously increases.

---

## 🎮 Play the Game

```

https://njansanjay.github.io/2D-Car-Race/⁠�
```

## ✨ Features

* 🛣️ 3-lane road system
* ⬅️➡️ Smooth left & right movement
* 🚗 Multiple enemy vehicles
* ⚡ Increasing difficulty over time
* 💯 Score tracking system
* 🏆 Persistent High Score (localStorage)
* 🔁 Restart system (Press R)
* ♻️ High score reset (Press H)
* 🎨 Clean UI with separate menu & game over screens

## 🧪 Controls 

* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Right
* ⏎ Enter → Start Game
* 🔁 R → Restart Game
* ♻️ H → Reset High Score

---

## 📈 Gameplay Mechanics

* Enemies spawn randomly across all 3 lanes
* At least one lane remains free to ensure playability
* Speed gradually increases for higher difficulty
* Collision ends the game


---

## ⚠️ Limitations

* No sound effects
* No mobile touch controls
* No pause feature
